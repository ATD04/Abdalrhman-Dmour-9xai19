# READ-ONLY Toward Source Environments

This Sandbox environment is strictly **READ-ONLY** and isolated from operational infrastructure.

### 1. Non-Intrusive Design
All data artifacts provided in this sandbox are derived from simulations (SUMO) or representative visual sources. The system does not write, send commands, or interact with real-world traffic signal controllers.

### 2. Operational Isolation
- **No Control Loop**: There is no feedback loop connected to the Wadi Saqra signal controllers.
- **Decision Support Only**: All recommendations generated by the system (Phase 2/3) are for human evaluation and decision support only.
- **Physical Isolation**: The sandbox logic runs in a virtualized container environment with no physical access to the municipal traffic network.

### 3. Data Integrity
Outputs are restricted to authorized users and are aligned with secure handling protocols for traffic logs and visual media.
