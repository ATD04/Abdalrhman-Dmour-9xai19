# JNPI Plans Folder

This folder contains product and implementation plans for the JNPI platform.

## Files
- `frontend_plan.md` — Full frontend roadmap (core + future features).
- `workflow_plan.md` — Workflow Service roadmap from MVP to production operations.
- `latency_audit_and_massive_reduction_plan.md` — Cross-service latency audit and reduction roadmap.
- `streaming_latency_deep_investigation_2026-03-26.md` — Deep root-cause analysis for the 10s streaming case and mitigation plan.

## Usage
- Treat these plans as living documents.
- Update acceptance criteria after each sprint.
- Keep implementation status synced with milestones in `app/docs/platform_milestones.md`.
